
# Clinic Management System

Welcome to the Clinic Management System


## Tech Stack

**Client:** ReactJS, Tailwind CSS

**Server:** NodeJS, Express

**Database:** MongoDB


## Installation

Install the project with npm

To install all dependencies on backend:
```bash
  cd backend
  npm install
```

## To Run the Backend:
```bash
npm run start-dev
```
